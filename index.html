üçπ Overview

JuiceToast is a lightweight, flexible, and dependency-free toast notification library for modern web apps.

  • Zero dependencies
  • ESM & UMD support
  • Queue handling, theme management
  • Dynamic toast types and backward compatibility

üì¶ Installation

ESM
import juiceToast from "https://cdn.kyrt.my.id/libs/js/juice-toast/1.1.0/juice-toast.esm.min.js";
UMD (Browser)
<link rel="stylesheet" href="https://cdn.kyrt.my.id/libs/css/fontic/2.0.0/juice-toast/style.min.css">
<script src="https://cdn.kyrt.my.id/libs/js/juice-toast/1.1.0/juice-toast.umd.min.js"></script>

üöÄ Quick Start

juiceToast.setup({
  success: { icon: "check", theme: "light", duration: 2000 },
  error: { icon: "x", bg: "#7f1d1d", color: "#fff", closable: true }
});

juiceToast.success("Operation completed successfully.");
juiceToast.error({ title: "Error", message: "Unexpected error." });

‚öôÔ∏è API Reference

setup(config)
juiceToast.setup({ success: { bg: "green" }, error: { bg: "red" } });
addType(name, config)
juiceToast.addType("warning", { bg: "#facc15", color: "#111" });
defineTheme(name, styles)
juiceToast.defineTheme("ocean", { bg: "#0ea5e9", color: "#fff" });
setTheme(name)
juiceToast.setTheme("dark");
clear()
juiceToast.clear();
destroy()
juiceToast.destroy();
